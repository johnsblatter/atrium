KDoc of Atrium

Atrium is split up in different modules. The packages shown below contain classes etc. of all modules excluding the once of atrium-imp-robstoll. Following a brief overview of the modules:

  • atrium-api defines the contracts of atrium

  • atrium-api-late-binding defines the contract for AtriumFactory and ThrowableFluent by providing dummy implementations which are eventually replaced by an atrium implementation.

    • atrium-impl-common provides common classes for implementations of atrium -- this module is deprecated and will be removed in the next version
    • atrium-impl-robstoll is an implementation of atrium and implements both, atrium-api as well as atrium-api-late-binding (hence provides a replacement for AtriumFactory and ThrowableFluent)
  • atrium-assertions contains all assertion functions as well as the ReporterBuilder.
  • atrium-spec contains Spek specifications for interfaces (defined in atrium-api) which can be reused by atrium implementations to assure they fulfill the specifications.
  • atrium-verb contains out of the box assertion verbs.
