Makes the assertion that AssertionPlant.subject is equal to expected with an error tolerance (range including bounds).

It effectively compares the absolute difference between AssertionPlant.subject and expected and compares it with the tolerance. As long as it is less than or equal the tolerance the assertion holds; otherwise it fails. A more mathematical way of expressing the assertion is the following inequality:
