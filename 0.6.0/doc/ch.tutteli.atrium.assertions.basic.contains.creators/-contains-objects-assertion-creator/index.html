ContainsObjectsAssertionCreator

abstract class ContainsObjectsAssertionCreator<T : Any, S, B : SearchBehaviour, C : Checker> : ContainsAssertionCreator<T, S, C>

Represents the base class for Contains.Creators which use bare objects as search criteria (matching them with ==).

It provides a template to fulfill the job of creating the sophisticated contains assertion.

Parameters

T - The type of the AssertionPlant.subject. S - The type of the search criteria. B - The type of the current Contains.SearchBehaviour. C - The type of the checkers in use (typically a sub interface of Contains.Checker).
