newTextListAssertionGroupFormatter

fun newTextListAssertionGroupFormatter(bulletPoints: Map<Class<out IBulletPointIdentifier>, String>, assertionFormatterController: IAssertionFormatterController, objectFormatter: IObjectFormatter, translator: ITranslator): IAssertionFormatter

Overrides IAtriumFactory.newTextListAssertionGroupFormatter

Creates an IAssertionFormatter which is intended for text output (e.g. for the console) and formats IAssertionGroups of type IListAssertionGroupType.

Parameters

bulletPoints - The bullet points used in reporting; will typically use the bullet point registered for IListAssertionGroupType as prefix for each IAssertion in IAssertionGroup.assertions. assertionFormatterController - The controller used to steer the flow of the reporting. objectFormatter - The formatter which is used to format objects other than IAssertions. translator - The translator which is used to translate ITranslatable such as IBasicAssertion.description.

Return
The newly created assertion formatter.
