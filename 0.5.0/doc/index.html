KDoc of Atrium

Atrium is built up by different modules. The packages shown below contain classes etc. of all modules excluding the one of atrium-imp-robstoll and atrium-imp-robstoll-lib.

Bundle modules:

  • atrium-cc-de_CH-robstoll defines to have a dependency on atrium-api-cc-de_CH, atrium-translations-de_CH and atrium-core-impl-robstoll
  • atrium-cc-en_UK-robstoll defines to have a dependency on atrium-api-cc-en_UK, atrium-translations-en_UK and atrium-core-impl-robstoll
  • atrium-cc-infix-en_UK-robstoll defines to have a dependency on atrium-api-cc-infix-en_UK, atrium-translations-en_UK and atrium-core-impl-robstoll

API modules:

  • atrium-api-cc-de_CH provides an assertion function API in German where the main focus of this API's design is put on ease of use/compatibility with code completion functionality of an IDE. It is a pure fluent API.
  • atrium-api-cc-en_UK provides an assertion function API in English where the main focus of this API's design is put on ease of use/compatibility with code completion functionality of an IDE. It is a pure fluent API.
  • atrium-api-cc-infix-en_UK provides an assertion function API in English where the main focus of this API's design is put on ease of use/compatibility with code completion functionality of an IDE. It is an infix API.

Assertion modules:

  • atrium-assertions contains all impl-functions and builder templates which can be used to create an API. Moreover it contains the ReporterBuilder.
  • atrium-translations-de_CH provides translations in German for Translatables used in assertion functions.
  • atrium-translations-en_UK provides translations in English for Translatables used in assertion functions.
  • atrium-verbs contains out of the box assertion verbs (we suggest to use your own assertion verbs).

Core modules:

  • atrium-core-api defines the contracts of the core of Atrium.
  • atrium-core-api-late-binding defines the contract for AtriumFactory by providing a dummy implementation which eventually needs to be replaced by a real implementation.
  • atrium-core-impl-robstoll provides an AtriumFactory which uses the implementations of atrium-impl-robstoll-lib
  • atrium-core-impl-robstoll-lib robstoll's implementation of the core of Atrium.

Misc modules:

  • atrium-spec contains Spek specifications for interfaces (defined in atrium-core-api) and assertion functions. The specifications can be reused by implementation of the core of Atrium as well as by API implementations to assure they fulfill the specifications.
