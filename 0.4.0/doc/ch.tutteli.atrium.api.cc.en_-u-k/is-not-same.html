isNotSame

fun <T : Any> IAssertionPlant<T>.isNotSame(expected: T): IAssertionPlant<T>

Makes the assertion that IAssertionPlant.subject is not the same instance as expected.

This method might enforce in the future, that expected has to be the same type as IAssertionPlant.subject. Currently the following is possible: assert(1).isNotSame(1.0)

Exceptions

AssertionError - Might throw an AssertionError if the assertion made is not correct.

Return
This plant to support a fluent API.
