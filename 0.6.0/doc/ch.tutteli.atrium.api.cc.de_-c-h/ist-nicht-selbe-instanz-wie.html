istNichtSelbeInstanzWie

fun <T : Any> Assert<T>.istNichtSelbeInstanzWie(expected: T): AssertionPlant<T>

Makes the assertion that AssertionPlant.subject is not the same instance as expected.

This method might enforce in the future, that expected has to be the same type as AssertionPlant.subject. Currently the following is possible: esGilt(1).istNichtSelbeInstanzWie(1.0)

Exceptions

AssertionError - Might throw an AssertionError if the assertion made is not correct.

Return
This plant to support a fluent API.
